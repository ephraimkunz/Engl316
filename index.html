Your First C Program

Introduction

This tutorial will teach you how to build and run your first computer program. It will use the programming language C, a common language that is used everywhere software is written. Your program will be simple, just printing a single line to the screen, but it will be the stepping stone for bigger and better programs.

Click here to see why learning to code is valuable.

List of Equipment/Materials

You'll need to get a few things ready before you're ready to start this tutorial:

  • Computer running the Ubuntu operating system (Not Mac or Windows)
  • Basic computer familiarity

Instructions

  1. Launch the program "Text Editor" that comes with your Ubuntu operating system.
Info: If you have another text editor installed that you prefer to use, go ahead. However, make sure you use one that won't change the characters you type to make them pretty. Microsoft Word or other word processors are not suitable for use as coding text editors.
If you're unfamiliar with Linux, follow these instructions to open up "Text Editor":
  1. Click on the "Launcher" icon in the upper right corner of the screen. It is found here:
  2. A search bar should open up, looking like this: Type Text Editor into it.
  3. Click the first application shown, which should be named "Text Editor."
  1. The text editor will launch with a new document. Save your new empty file as helloworld.c to the Desktop.
    2. Warning: Computers can be very nitpicky about spacing, punctuation, and capitalization. For all the text you will type or copy and paste in this tutorial, make sure that you match our spacing, punctuation, and capitalization exactly.
    If the computer doesn't do what you expected when you type a command, double check your typing!
  2. In the helloworld.c file open in "Text Editor," type or copy and paste the following code: #include <stdio.h> int main() { printf("Hello World!\n"); return (0); } Click here for an explanation of what the program you just typed will do.
  3. Save helloworld.c to the Desktop.
  4. Open the Bash terminal by pressing ctrl+alt+t. Learn more about the terminal here.
  5. In the terminal window, type cd Desktop and press enter to navigate to the Desktop. The prompt the shell gives you will change to show that you are now on the Desktop.
  6. In the terminal, type gcc helloworld.c and press enter to compile the program. If everything worked, nothing should be printed on the screen except for a new prompt. Learn more about gcc here.
  7. In the terminal, type ./a.out and press enter to run the program. If everything worked, you should see the words "Hello World!" printed on a line by themselves in the shell. Learn more about what you just ran here.

Conclusion

Congratulations! You have written your first C program. Although humble, this program shows all the necessary steps for writing, compiling and running a program. For more convenient tutorials on programming in C, visit this website.
