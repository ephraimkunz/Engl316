Your First C Program

Introduction

This tutorial will teach you how to build and run your first computer program. It will use the programming language C, a common language that is used everywhere that software is written. Your program will be simple, just printing a few lines to the screen. But it is the stepping stone for bigger and better programs. Click here to see why learning to code is valuable.

List of Equipment/Materials

  • Computer running the Ubuntu operating system (Not Mac or Windows)
  • Basic computer familiarity

Instructions

  1. If you don't already have one installed, install a text editor. We will demonstrate using the default program "Text Editor" on Ubuntu.
    2. Warning: Microsoft Word or other word processors are not suitable for use as coding text editors.
  2. Launch the text editor you installed in Step 1.
  3. The text editor will launch with a new document. Save your new empty file as helloworld.c to the Desktop.
  4. In the helloworld.c file, type the following: 

                            
                        #include <stdio.h>

                        int main() {
                            printf("Hello World!\n");
                            return (0);
                        }
    Click here to understand what the lines you just typed will do.
    Warning: Be sure that you type the above text exactly as written. Missing or adding any extra punctuation will cause your program to not compile.
  5. Save helloworld.c
  6. Open the Bash terminal by pressing ctrl+alt+t. Learn more about the terminal here.
  7. In the terminal window, type cd Desktop and press enter to navigate to the Desktop.
  8. In the terminal, type gcc helloworld.c to compile the program. Learn more about gcc here.
  9. In the terminal, type ./a.out to run the program. Learn more about what you just ran here.

Conclusion

Congratulations! If everything worked, you should now see the text "Hello World!" printed inside the terminal. You have written your first C program. For more convenient tutorials on programming in C, visit this website.
